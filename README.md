# Zeldomizer

The Legend of Zelda ROM manipulation library.

- Zeldomizer
  - Modifies aspects of the Legend of Zelda ROM
- Breadbox
  - Emulates a 6502 processor at the cycle level
- Disaster
  - Assembles and disassembles 6502 code

## Implemented

- Edit character dialog
- Edit ending text
