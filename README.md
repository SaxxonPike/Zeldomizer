# breadbox

MOS and (future) Commodore chip emulation in F#. These cores are designed to be accurate on the cycle level.

### Complete:

- CPU: 6502 (all opcodes tested)

### Planned:

- CPU: other 6502 variants from MOS including 6510
- SID: 6581 and 8580
- VIC II: 6567, 6569, 6572
- VIA: 6522
- CIA: 6526
